# Car Rendering

**Project:** Car Rendering with Future Development  
**Topic:** Displaying a list of car models in a React 19 app and laying groundwork for further features

---

## 🇬🇧 Project Overview

Car Rendering is a small React application that demonstrates how to:
1. Create a static array of car models  
2. Render that array to a `<ul>` list using the `map()` method  
3. Use React 19’s `createRoot` API for mounting  
4. Provide a minimal boilerplate you can extend later (forms, API calls, styling, state management, etc.)

The goal is to give a starting point for further enhancements, such as:  
- Adding dynamic CRUD (Create / Read / Update / Delete)  
- Integrating a Go (or any) backend  
- Building more advanced UI components (tables, filters, sorting)  
- Applying CSS frameworks like TailwindCSS
